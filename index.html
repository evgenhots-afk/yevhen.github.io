Про мене

Я — Євген, досвідчений веб-розробник з багаторічним стажем. Маю глибокі знання в HTML5, CSS3, JavaScript та сучасних фреймворках, таких як React. Моя пристрасть — створювати інтуїтивно зрозумілі, ефективні та візуально привабливі веб-додатки. Я завжди прагну вдосконалювати свої навички та слідувати останнім тенденціям у галузі розробки.

Проєкти

E-commerce Marketplace

Масштабований інтернет-магазин, розроблений на React з використанням Redux для управління станом. Включає систему автентифікації користувачів, кошик покупок та інтеграцію платіжного шлюзу.

Personal Blog Platform

Платформа для блогів з динамічним контентом, створена на Node.js та Express.js. Має зручний інтерфейс для адміністрування, систему коментарів та можливість завантаження медіафайлів.

Task Management App

Інтуїтивно зрозумілий додаток для управління завданнями, який дозволяє створювати, редагувати та відстежувати прогрес. Розроблено з використанням Vue.js та Firebase для зберігання даних у реальному часі.
